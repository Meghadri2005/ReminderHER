# ReminderHER
To run the app follow these steps:

1. clone the repository
2. Install libraries
```bash
pip install -r requirements.txt
```
3. make the secret.py and follow those steps
4. make a virtual environment by this command
```bash
    python -m venv venv
```
5. Activate it
```bash
venv/Scripts/activate
```

6. Run the app
```bash
python app.py
```
